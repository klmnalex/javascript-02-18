### 1.1.3 JavaScript as an interpreted language
interpreted (parancsértelmezős) language:
- JavaScript (böngészőbe épített JS engine)
- Python (python.exe)
- PHP (index.php)

compiled (fordítós) languages:
- Java
- C#
- C/C++
